# Portfolio-U--Collection
A collection of modern resume and portfolio templates built with pure HTML and CSS, featuring biography cards, skill sections, educational timelines, and fully custom UI designs.

### Multi-Template Resume & Portfolio UI Collection

This project is a complete set of custom-designed resume and portfolio templates created using pure HTML and CSS.
It includes multiple biography-style profile cards inspired by public figures such as Mark Zuckerberg, Jeff Bezos, and Bill Gates, along with professional CV layouts, skill sections, and UI components.

Each template highlights key resume sections such as:

Personal Information

Education

Skills

Achievements

Biography

Professional Experience

Social Media Profiles

Circular Skill Meters

Award / Reference Sections

These templates are useful for:

Personal portfolios

Student projects

Developer CVs

UI/UX practice

Front-end layout learning

GitHub showcases
